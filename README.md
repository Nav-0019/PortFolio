# Portfolio Website

A personal portfolio website showcasing my projects, skills, and experience. Built with HTML, CSS, and JavaScript, featuring a modern, responsive design.

## Features

- **Modern UI**: Clean, responsive design with glassmorphism effects and smooth animations.
- **Project Showcase**: Detailed sections for major and minor projects.
- **Skills Section**: Highlighted technical skills and tools.
- **Coding Stats**: Integration of LeetCode stats to showcase problem-solving abilities.
- **Contact Form**: Easy way to get in touch.

## Projects Included

### Major Projects
1.  **Loan Risk Prediction**: A machine learning model to predict loan default risk.
2.  **Portfolio Website**: This website itself!
3.  **COVID-19 Trend Analysis**: Data analysis and visualization of pandemic trends.

### Minor Projects
- Isekai Gaming Webapp
- To-Do List
- Calculator
- Student Score Analyzer

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Custom CSS with Flexbox/Grid, CSS Variables, and Animations.
- **Fonts**: Google Fonts (Poppins).

## How to Run

1.  Clone the repository:
    ```bash
    git clone https://github.com/Nav-0019/Portfolio.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd Portfolio
    ```
3.  Open `index.html` in your browser.
